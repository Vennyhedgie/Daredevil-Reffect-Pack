# Havoc Specialist

While in combat, if the trait Havoc Specialist is equipped and endurance is at full, an extra indicator over your endurance will appear to remind you of uptime. If you don't like this, you can disable the Havoc Specialist element in the pack.

<img width="875" height="203" alt="image" src="https://github.com/user-attachments/assets/388a5fc8-5c20-40eb-8a1c-b05b53291dcf" />


# How to use
Simply download the zip file to your reffect root folder like so:

<img width="481" height="223" alt="image" src="https://github.com/user-attachments/assets/ba6eea63-a033-4a74-849a-b1d14e3c12fe" />

Then right click > Extract here (the necessary files will be added to the packs and icons subfolders).

Then, return to reffect ingame and refresh packs.
Finally, enable whichever interface size you play with. Normal is enabled by default:
